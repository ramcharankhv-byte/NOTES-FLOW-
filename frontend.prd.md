# NOTES-FLOW FRONTEND MASTER PROMPT

Build a modern futuristic SaaS frontend called **Notes-Flow**.

Tech Stack:

- React + Vite
- Tailwind CSS
- Framer Motion
- React Router
- Axios
- Zustand or Context API for state management
- GSAP for advanced scroll animations
- Three.js or React Three Fiber for 3D elements
- Lucide React Icons
- Shadcn UI components

Theme:

- Full dark futuristic UI
- Background should be matte black
- Center radial gradient splash dark blue glow
- Glassmorphism cards
- Neon blue highlights
- Smooth transitions
- Rounded corners
- Premium SaaS feel
- Apple + Linear + Vercel inspired design
- Heavy use of motion and hover animations
- Responsive on all devices
