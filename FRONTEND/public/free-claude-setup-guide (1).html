// Windows — CMD & PowerShell Setup
01
Install Git
Download and install Git for Windows. After installation, close and reopen CMD.
Browser
https://git-scm.com/download/win
✓ verify
git --version
02
Install UV (Python Package Manager)
Open CMD as Administrator, then run the command below. Close and reopen CMD after.
CMD — Run as Administrator 
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
✓ verify
uv --version
03
Install Python 3.14 via UV
CMD 
uv python install 3.14
✓ verify
python --version
04
Clone the NVIDIA NIM Project
CMD 
cd %USERPROFILE%\Desktop
git clone https://github.com/Alishahryar1/free-claude-code.git nvidia-nim
cd nvidia-nim
05
Create .env File
Copy the example environment config file.
CMD
CMD 
copy .env.example .env
OR PowerShell
PowerShell 
Copy-Item .env.example .env
06
🔑 Create NVIDIA Account & Get API Key
You need a free NVIDIA account to get your NIM API key.
⚠ This step is required. Without an API key, the server will not work.
Step 1 — Open in Browser
https://build.nvidia.com/  →  Sign Up (free account)
Step 2 — Get API Key
Profile → API Keys → Generate Key → Copy it
Step 3 — Paste into .env file 
# Open .env file in Notepad:
notepad .env

# Find this line and paste your key:
NVIDIA_API_KEY=your_nvidia_api_key_here
✓ Save and close Notepad after pasting your key.
07
Install Dependencies
CMD 
uv sync
08
Start NVIDIA NIM API Server
⚠ Keep this terminal window open always. Do NOT close it.
CMD — Terminal 1 
uv run uvicorn server:app --host 0.0.0.0 --port 8082
✓ You should see: Uvicorn running on http://0.0.0.0:8082
09
Install Claude Code
Open a NEW CMD window for this step.
CMD — New Window 
curl -fsSL https://claude.ai/install.sh | bash
If you see 'bash' is not recognized, use this instead:
CMD — Git Bash Fallback 
"C:\Program Files\Git\bin\bash.exe" -c "curl -fsSL https://claude.ai/install.sh | bash"
✓ verify
claude --version
10
🚀 Run Claude Code with NVIDIA NIM
Run these commands every time you want to start Claude.
CMD Version
CMD — Terminal 2 
set ANTHROPIC_AUTH_TOKEN=freecc
set ANTHROPIC_BASE_URL=http://localhost:8082
claude
PowerShell Version
PowerShell — Terminal 2 
$env:ANTHROPIC_AUTH_TOKEN="freecc"
$env:ANTHROPIC_BASE_URL="http://localhost:8082"
claude
11
Recommended Workflow (Every Session)
Terminal 1 — Keep Running
uv run uvicorn server:app \
  --host 0.0.0.0 \
  --port 8082
Terminal 2 — Claude
set ANTHROPIC_AUTH_TOKEN=freecc
set ANTHROPIC_BASE_URL=http://localhost:8082
claude
12
Optional: One-Click Start File (.bat)
Create start-claude.bat inside the nvidia-nim folder and paste the content below.
start-claude.bat 
@echo off
start cmd /k "uv run uvicorn server:app --host 0.0.0.0 --port 8082"
timeout /t 5
set ANTHROPIC_AUTH_TOKEN=freecc
set ANTHROPIC_BASE_URL=http://localhost:8082
claude
ℹ Double-click start-claude.bat to launch everything automatically.
13
Common Errors & Fixes
Error: 'uv' is not recognized
Restart CMD after installing UV.
Error: 'claude' is not recognized
Restart terminal after Claude install.
Error: Port 8082 already in use
Find PID 
netstat -ano | findstr :8082
Kill Process 
taskkill /PID PID_NUMBER /F

Optional: One-Click Start Script
Create the script file:
Terminal 
nano ~/Desktop/nvidia-nim/start-claude.sh
Paste this content:
start-claude.sh 
#!/bin/bash
cd ~/Desktop/nvidia-nim
uv run uvicorn server:app --host 0.0.0.0 --port 8082 &
sleep 5
ANTHROPIC_AUTH_TOKEN="freecc" \
ANTHROPIC_BASE_URL="http://localhost:8082" \
claude
Save: CTRL+O → Enter → CTRL+X, then make it executable:
Terminal 
chmod +x ~/Desktop/nvidia-nim/start-claude.sh
./start-claude.sh
